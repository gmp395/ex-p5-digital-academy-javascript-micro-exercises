# ex-p5-digital-academy-javascript-micro-exercises

### Lista de Ejercicios

1. **Ejercicio 1:** Escribe un programa que escriba en la pantalla un texto de tipo que diga "Hello Javascript".
2. **Ejercicio 2:** Escribe un programa que pida el nombre del usuario con un input y escriba un texto que diga "Hola".
3. **Ejercicio 3:** Escribe un programa que pida 3 números y escriba en la pantalla el mayor de los tres.
4. **Ejercicio 4:** Escribe un programa que pida una frase y escriba cuántas veces aparece la letra a.
