EX - P5 - Javascript - Micro Exercises
Ex - 01
Escribe un programa que escriba en la pantalla un texto de tipo <h1> que diga Hello Javascript.

Ex - 02
Escribe un programa que pida el nombre del usuario con un input y escriba un texto que diga Hola <nombre-de-usuario>

Ex - 03
Escribe un programa que pida 3 números y escriba en la pantalla el mayor de los tres.

Ex - 04
Escribe un programa que pida una frase y escriba cuantas veces aparece la letra a

Ex - 05
Escribe un programa que pida una frase y escriba las vocales que aparecen. Una vocal repetida cuenta como una.

Ex - 06
Escribe un programa que pida dos números, que el usuario pueda elegir suma o resta y escriba “La suma de con es ” o “La resta de con es ”.

Ex - 07
Escribir un programa que escriba en pantalla los divisores comunes de dos números dados

Ex - 08
Escribir un programa que nos diga si un número dado es primo (no es divisible por ninguno otro número que no sea él mismo o la unidad)

Ex - 09
Crea un programa que pida el nombre de usuario y la edad. Si es mayor de 18 el programa debe imprimir en pantalla <nombre> puedes conducir. Y recuerda: Si bebes no conduzcas. Si el usuarió es menor de edad entonces se deberá mostrar <nombre>, mejor te compras una bici. La impresión debe hacerse en un elemento oculto que se mostrará al validar los datos de usuario. Se deberá poder volver a ocultar el elemento mediante alguna acción.

Ex - 10
Solicita el nombre del alumno/a y la nota (número) al profesor y imprime en pantalla una tabla clasificación con nombre y calificación. Calificación según la nota:

0-3: Muy deficiente
3-5: Insuficiente
5-6: Suficiente
6-7: Bien
7-9: Notable
9-10: Sobresaliente
Ejemplo de Output
Nombre	Calificación
Pepito	Insuficiente
Pepita	Notable
